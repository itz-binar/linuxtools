## 🔧 Installation

1. Clone the repository:
```bash
git clone https://github.com/itz-binar/linuxtools.git
cd linuxtools
```

2. Make the script executable:
```bash
chmod +x scripts/install-kali-termux.sh
```

3. Run the installer:
```bash
./scripts/install-kali-termux.sh
```

## 📚 Usage

### Starting Environments
```bash
# Start Kali Linux
kali

# Start Kali as root
kali-root

# Start Arch Linux
arch

# Start Arch as root
arch-root
```

### Common Commands
```bash
# Update system
update

# View processes
htop

# Start SSH server
sshd

# Access Python tools
python
pip
```
